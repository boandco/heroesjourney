# journey10x
Website for JOURNEY 10X


## Set up
Run `bundle install && npm install`

## Local development
Run `npm run serve` to build the local server. Navigate to [http://127.0.0.1:4000/](http://127.0.0.1:4000/).


To edit the content on the site, visit `/admin`.
